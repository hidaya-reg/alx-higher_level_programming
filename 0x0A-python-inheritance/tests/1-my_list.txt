=================================
The ``1-my_list.py test`` module
=================================

Usage
=====
``def print_sorted(self)``: that prints the list, but sorted
				(ascending sort)
::

>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(3)
>>> my_list.append(10)
>>> my_list.append(5)
>>> my_list.append(4)
>>> my_list.append(11)
>>> print(my_list)
[1, 3, 10, 5, 4, 11]
>>> my_list.print_sorted()
[1, 3, 4, 5, 10, 11]
>>> print(my_list)
[1, 3, 10, 5, 4, 11]
